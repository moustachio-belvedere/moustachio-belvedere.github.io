This site is a portfolio website written for the Coursera course Web Design for Everybody Capstone. It is intended to be responsive and accessible.

The Image Showcase page features a responsive image gallery with a selection of images from Wikipedia. The Javascript Showcase features an interactive SVG canvas developed using the Javscript library D3.js. The WebGL Showcase is a simple version of Conway's Game of Life rendered on your graphics card using WebGL. Lastly, the Licensing Credits is a listing of all external code/images used in this website and their associated licensing information.
